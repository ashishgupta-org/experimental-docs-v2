<Steps>
  <Step title="Create a Workspace">
    Set up a workspace for your team. A workspace provides a dedicated environment for managing projects, datasets, policies, and telemetry.
  </Step>
  <Step title="Set Up Your First Project">
    Create a project to manage the evaluation lifecycle for an AI agent or application. Projects organize runs, datasets, policies, metrics, and dashboards for a single initiative. 
  </Step>
  <Step title="Configure Business Baselines">
    Define the metrics that represent success for your agent—such as cost per interaction, success rate, error rate, or efficiency. Baselines provide the thresholds used to measure performance over time.
  </Step>
  <Step title="Integrate Your Agent">
    Use the AMP SDK to connect your agent and send telemetry (sessions and traces) into the platform. After integration, real-time activity appears in the project's Runs and Traces views.
  </Step>
  <Step title="Create Policies">
    Define governance rules to automate evaluation of your agent's behavior. Policies can validate correctness, safety, compliance, or business rules. You can run them manually, schedule them, or attach them to datasets for consistent regression testing.
  </Step>
  <Step title="Monitor Performance">
    Use dashboards and project views to track performance trends. Monitor sessions, review trace waterfalls, analyze errors, and validate improvements as policies detect issues or regressions.
  </Step>
</Steps>

## Platform Navigation

The interface has two levels, each with its own navigation structure.

| Level | Navigation |
|---|---|
| **Workspace** | Top bar with workspace switcher, projects list, notifications, and support menu. No sidebar. Access workspace-wide settings and team administration. |
| **Project** | Left sidebar for project navigation. Top bar shows contextual breadcrumbs for the active project. |


## User Roles

| Role | Description |
|---|---|
| **Workspace Admin** | Assigned Account Admin by default. Can create and manage projects. |
| **Workspace Member** | Any invited user. Can also create new projects. |

## Profile and Workspace Access

Click your avatar and name in the bottom-left corner to access the following options:

| Option | Description |
|---|---|
| **Switch Workspace** | Move to a different workspace—each has isolated data, separate access control, and independent billing. |
| **Invite Team Members** | Add users to your current workspace. |
| **View Profile** | Access and update your profile settings. |
| **Log Out** | End your session and return to the login screen. |

### Switch Workspace

If you belong to multiple workspaces, switch context to view a different client or team's projects.

![Switch workspace](/agent-management-platform/images/switch-workspace.png)

1. Click your avatar in the bottom-left corner.
2. Select **Switch Workspace**, then select the workspace.
