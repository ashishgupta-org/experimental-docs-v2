The XO Platform provides built-in intelligence features that help your AI Agent handle the natural twists and turns of human conversation — including interruptions, context shifts, and emotional cues — without losing track of the user's goals.

---

## Dialog Management

| Feature | Description |
|---|---|
| **Interruptions and context switching** | Control hold and resume behavior at the app, task, and node levels. [Learn more](./conversation-management/manage-interruptions.md) |
| **Entity value modification** | Let users amend entity values mid-conversation and control post-amendment behavior. [Learn more](../natural-language/nlu-configurations/amend-entities.md) |
| **Multi-intent detection** | Detect and execute multiple intents from a single user utterance. [Learn more](../natural-language/nlu-configurations/multi-intent-detection.md) |
| **Ignore words** | Mark words and phrases that don't aid task recognition so the AI Agent can respond faster and more accurately. [Learn more](./conversation-management/ignore-words-and-field-memory.md) |

---

## Context Management

AI Agents are stateless by default, but conversations depend on shared context. The Platform lets you capture and reuse contextual data across intents, entities, and FAQs to create more natural, human-like experiences.

| Feature | Description |
|---|---|
| **Context switching** | Enable or disable context switches between tasks and pass contextual data between them. [Learn more](context-management.md) |
| **Session variables** | Use Platform-provided or custom session variables with defined scope. [Learn more](../use-cases/using-session-and-context-variables.md) |

---

## Sentiment Management

Trigger conditional flows and steer conversations based on a user's emotional state.

| Feature | Description |
|---|---|
| **Sentiment analysis** | Detect and respond to user emotions to deliver better experiences. [Learn more](./sentiment-management/sentiment-management-overview.md) |
| **Tone processing** | Analyze tone scores and use them to drive dialog transitions (for example, escalating to a live agent). [Learn more](./sentiment-management/tone-analysis.md) |
