Contact Center AI reports allow supervisors to assess agent performance against key metrics such as average handle time. Choose from the following report types:

| Report | Description |
|--------|-------------|
| [Agent Activity Summary Report](../reports/agent-activity-summary-report.mdx) | A daily summary of agent time by status. |
| [Agent Chat Metrics Report](../reports/agent-chat-metrics-report.mdx) | A daily summary of agent performance metrics relevant to chat conversations. |
| [Agent Login Logout Summary Report](../reports/agent-login-logout-summary-report.mdx) | A detailed report of an agent's login and logout events. |
| [Agent Metrics Daily Report](../reports/agent-metrics-daily-report.mdx) | A daily summary of agent performance metrics, by channel. |
| [Agent Status Detail Report](../reports/agent-status-detail-report.mdx) | Comprehensive information on all status changes that occurred within a date range. |
| [Dialer Campaign History Report](../reports/dialer-campaign-history-report.mdx) | Records all dialer attempts by all agents for selected campaigns, during a time interval. |
| [Interaction Details Report](../reports/interaction-details-report.mdx) | A comprehensive report covering how all interactions were processed, by whom, and their final dispositions. Includes conversation metadata excluding recordings or transcripts. |
| [IVR Containment Report](../reports/ivr-containment-report.mdx) | Covers all calls made, callers identified, and per-intent data and outcomes on a daily basis. Requires predefined message tags in the bot designer. Applicable for voice calls only. |
| [Queue Dispositions Report](../reports/queue-dispositions-report.mdx) | Summarizes disposition codes for interactions associated with each queue. |
| [Queue Metrics Summary Report](../reports/queue-metrics-summary-report.mdx) | A daily report showing queue load and performance summary. |
| [Queue Chat Metrics Interval Report](../reports/queue-chat-metrics-interval-report.mdx) | Summarizes queue performance metrics relevant to chat conversations, aggregated over configurable intervals (30-minute default). |
| [Selected Hours Report](../reports/selected-hours-report.mdx) | Shows daily totals of every agent's productive hours, determined by built-in and custom statuses selected by users. |
| [Total Volumes Report](../reports/total-volumes-report.mdx) | Provides comprehensive statistics on conversations including counts and duration, categorized by channel and direction, within a specified time frame. |
| [Queue Metrics Interval Report](../reports/queue-metrics-interval-report.mdx) | Provides queue performance metrics at sub-daily intervals (15 minutes to 4 hours), including service level data highlighting met and unmet targets. |
| [Interaction Details by Segment](../reports/interaction-details-by-segment.mdx) | A comprehensive report covering how all interactions were processed for each segment. |
| [Skill Metrics Daily Report](../reports/skill-metrics-daily-report.mdx) | A daily summary of performance based on conversation skills, grouped by Skill and Day with no grouping by channel or direction. |
| [Latency Report](../reports/latency-report.mdx) | Highlights calls exceeding a defined latency threshold, such as all calls where P90 end-to-end latency is greater than 500 ms. |
