The Queue Dispositions Report is a detailed report on sessions and duration for each code in a disposition set.

---

## Generating the Report

1. On the Reports tab, select **New Report**.
2. In the list of reports, choose **Queue Dispositions Report**.
3. Enter the **Report Name** and **Report Description**.
4. Select a **Time Period**.
5. Select **Data Filters**.
6. Select **Report Generation** and choose from the following options:
   - **Run On Demand**
   - **Schedule** — You can create up to six schedules (only one if you select Hourly).

     ![Schedule Options](../images/schedule-options.png)
7. Select **Format**:
   - PDF
   - CSV
8. Enter a **Send To** address (single email address or mailing list).
9. Select **Create**.
10. The Queue Dispositions Report is generated.

    ![Queue Dispositions Report Created](../images/queue-dispositions-report-created.png)

11. Select the **Run** icon next to the report instance.

    ![Run Queue Dispositions Report](../images/run-queue-dispositions-report.png)

**Run On Demand**: Triggers report generation immediately when selected.

**Schedule**: Sets the report to generate automatically on the scheduled day and time, without needing to select Run.

12. Select the **PDF/Page** icon to download the report.

    ![Queue Dispositions Report PDF](../images/pdf-queue-dispositions-report.png)

<Note>The red dot beside the report indicates it was generated within the last 24 hours. You can download, save, and view the report in PDF format.</Note>

The report is generated and available for download.

![Queue Dispositions Report Output](../images/queue-dispositions-report-output.png)

## Report Fields

| Field | Description | Example | Total |
|-------|-------------|---------|-------|
| Disposition Code | Each disposition code in a disposition set. | Resolved | NA |
| Sessions | Number of conversations assigned this code. | 123 | SUM |
| % Total | Number of conversations assigned this code / Total conversations. | 83 | NA |
| Call Length – Total | Total length of all conversations assigned this code. | "00:45:52" | SUM |
| Call Length – Average | Total length / Number of sessions. | "00:45:52" | AVG |
| Call Length – % Total | Total length for this code / Total length for this set. | 23 | NA |
