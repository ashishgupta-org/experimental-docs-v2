The Agent Login Logout Summary Report is a detailed report of an agent's login and logout events.

---

## Generating the Report

1. On the Reports tab, select **New Report**.
2. In the list of reports, select **Agent Login Logout Summary Report**.
3. Enter the **Report Name** and **Report Description**.
4. Select a **Time Period**.
5. Select **Data Filters**.
6. Select **Report Generation** and choose from the following options:
   - **Run On Demand**
   - **Schedule** — You can create up to six schedules (only one if you select Hourly).

     ![Schedule Options](/ai-for-service/analytics/contact-center/reports/images//schedule-options.png)
7. Select **Format**:
   - PDF
   - CSV
8. Enter a **Send To** address (single email address or mailing list).
9. Select **Create**.
10. The Agent Login Logout Summary Report is generated.

    ![Agent Login Logout Summary Report Created](/ai-for-service/analytics/contact-center/reports/images//agent-login-logout-report-created.png)

11. Select the **Run** icon to generate the report.

    ![Run Agent Login Logout Summary Report](/ai-for-service/analytics/contact-center/reports/images//run-login-logout-summary-report.png)

**Run On Demand**: Triggers report generation immediately when selected.

**Schedule**: Sets the report to generate automatically on the scheduled day and time, without needing to select Run.

12. Select the **PDF/Page** icon to download the report.

    ![PDF Agent Login Logout Report](/ai-for-service/analytics/contact-center/reports/images//pdf-agent-login-logout-summary.png)

<Note>The red dot beside the report indicates it was generated within the last 24 hours. You can download, save, and view the report in PDF format.</Note>

The report is generated and available for download.

![Agent Login Logout Summary Report Output](/ai-for-service/analytics/contact-center/reports/images//agent-login-logout-summary-report-output.png)

## Report Fields

| Field | Description | Example |
|-------|-------------|---------|
| Login | DateTime at which the agent logged in, starting a new session. | "01/06/23 15:57:17" |
| Logout | DateTime at which the agent logged out, ending a session. | "01/06/23 15:57:17" |
| Duration | Duration of the session (Logout - Login). | "64:39:22" |
