The Agent Status Detail Report provides detailed information on all status transitions that occurred within a date range.

---

## Generating the Report

1. On the Reports tab, select **New Report**.
2. In the list of reports, choose **Agent Status Detail Report**.
3. Enter the **Report Name** and **Report Description**.
4. Select a **Time Period**.
5. Select **Data Filters**.
6. Select **Report Generation** and choose from the following options:
   - **Run On Demand**
   - **Schedule** — You can create up to six schedules (only one if you select Hourly).

     ![Schedule Options](/ai-for-service/analytics/contact-center/reports/images//schedule-options.png)
7. Select **Format**:
   - PDF
   - CSV
8. Enter a **Send To** address (single email address or mailing list).
9. Select **Create**.
10. The Agent Status Detail Report is generated.

    ![Agent Status Detail Report Created](/ai-for-service/analytics/contact-center/reports/images//agent-status-detail-report-created.png)

11. Select the **Run** icon next to the report instance.

    ![Run Agent Status Detail Report](/ai-for-service/analytics/contact-center/reports/images//run-agent-status-detail-report.png)

**Run On Demand**: Triggers report generation immediately when selected.

**Schedule**: Sets the report to generate automatically on the scheduled day and time, without needing to select Run.

12. Select the **PDF/Page** icon to download the report.

    ![Agent Status Detail Report PDF](/ai-for-service/analytics/contact-center/reports/images//pdf-agent-status-detail-report.png)

<Note>The red dot beside the report indicates it was generated within the last 24 hours. You can download, save, and view the report in PDF format.</Note>

The report is generated and available for download.

![Agent Status Detail Report Output](/ai-for-service/analytics/contact-center/reports/images//agent-status-detail-report-output.png)

## Report Fields

| Field | Description | Example |
|-------|-------------|---------|
| Begin | DateTime at which the agent begins their stay in a status. | "01/06/23 15:57:17" |
| End | DateTime at which the agent ends their stay in a status. | "01/06/23 15:57:17" |
| Standard Status | Status according to the standard statuses in the system. | Busy |
| Custom Status | Status according to the custom statuses configured by the customer. | Security Training |
| Duration | Time for which the agent's status was in the given state. | "323:21:23" |
