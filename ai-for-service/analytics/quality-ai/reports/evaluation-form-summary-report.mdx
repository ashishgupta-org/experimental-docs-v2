This report summarizes metric-level adherence across selected evaluation forms, providing aggregated compliance statistics and highlighting adherence rates and fatal metric violations.

It includes channel (Voice or Chat) and contact direction (Inbound/ or Outbound or All Directions), along with evaluated counts, adherence percentages, and failure trends to compare compliance across queues, channels, and interaction types.

## Generate Evaluation Form Summary Report

1. **Create the New Report**

    1. On the **Reports** tab, select **+ New Report**.

    1. From the **Create New Report** list, select the **Evaluation Form Summary Report**.
        ![Report Selection](/ai-for-service/analytics/quality-ai/reports/images/evaluation-form-report.png)

    1. Select **Next**.

1. Complete the **General** settings.

    1. Enter a unique **Name** for the report.

    1. Enter a **Description**.

    1. Select **Next**.

        ![General Settings](/ai-for-service/analytics/quality-ai/reports/images/create-new-report-general.png)

1. Complete the **Configuration** — **Filters and Options**.

    1. Select an **Evaluation Form** to define the data scope (at least one form is required).

    1. Select a **Date** or date range.

    1. Select a **Timezone**.

    1. Configure the required Filter.
    
        1. Select the required filter options such as **Channel** (**Voice** or **Chat**) with Contact Direction (**Inbound** or **Outbound** or **All Directions**), **Queue**, **Language**, **Agents**, **Agent Groups**, and **Duration Status** (evaluated, below threshold, and duration threshold). 
    
        1. Set the corresponding values for the selected/added filters. 

    1. Select **Optional Fields** to add additional columns to the report export. 

    1. Enter one or more addresses in the **Send Email to** field to send the automated reports (minimum one required). 

    1. Select **Next**.
    
        ![Configure Filters](/ai-for-service/analytics/quality-ai/reports/images/create-new-report-config.png)

1. Complete the **Schedule** to generate a report with the current configuration.

    1. Choose **Run on Demand** or **Schedule**.

        - **Run on Demand**: Generates the report immediately.
        - **Schedule**: Generates the report automatically.

    1. If you select **Schedule**:

        1. Choose a **Frequency or Repeat** (Hourly, Daily, Weekly, Monthly, Quarterly, or Yearly).

        1. Select a **Start Date** and **Timezone**.

        1. Select **Save**.

            <Note>This report includes only evaluated interactions in all aggregated metrics. Interactions below the duration threshold are not included in any aggregated metrics.</Note>

## Report Fields

### Metric Adherence Summary Report

Summarizes metric-level adherence across selected evaluation forms for eligible evaluated interactions. Includes total and applicable interactions, adhered interactions, adherence percentage, fatal metric flags, and contact direction breakdown (Inbound or Outbound or Combined) for compliance comparison.

Only interactions with `duration_status = evaluated` are included in calculations; `below_threshold` and `duration_unavailable` are excluded from all adherence aggregates.

<Note>When the **Contact Direction** filter is applied, the report aggregates only interactions matching the selected direction.</Note>

| Speech Crosstalk                        | Both              |        0 |                  3 |                             3 |                          2 |                       66.67 |

### View and Download the Report

1. On the **Reports** tab, locate the report in the **Actions** column.

1. Select More options (⋮), then select **History**.

1. Select **Download** to save the report as a CSV file.

### Report Format

- **Format**: CSV file
- **File Name**: [ReportName]-YYYY-MM-DD-HH-MM-SS.csv
