The Reports module includes three pre-defined reports, each tailored for specific analytics and monitoring needs. You can create, configure, and schedule automated reports based on these templates. All reports are delivered via email as CSV files and offer extensive filtering and customization options.

| Report Type | Description |
|---|---|
| [Agent Performance Report](../reports/agent-performance-report.mdx) | Exports detailed agent performance data based on selected scorecards or evaluation forms, including scorecard scores, attribute ratings, metric adherence, and coaching assignments. |
| [Interaction Evaluation and Conversation Analytics Report](../reports/interaction-conversation-analytics-report.mdx) | Exports comprehensive conversation data based on selected evaluation forms, providing detailed insights into interactions such as sentiment analysis, topics discussed, agent performance metrics, and evaluation scores. |
| [Evaluation Form Summary Report](../reports/evaluation-form-summary-report.mdx) | Summarizes evaluation metric adherence using selected forms, providing aggregated compliance statistics and highlighting metric-level performance, adherence rates, and fatal metric violations. |
