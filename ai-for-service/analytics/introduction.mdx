The Analytics section of the Kore.ai Platform provides dashboards to analyze real-time data generated by AI Agents during interactions.

## Available Dashboards

- **Summary Dashboard**: Summarizes the Conversations Dashboard, Users Dashboard, and Performance Dashboard.
- **Conversations Dashboard**: Provides insights into usage and containment of conversation sessions. Shows whether requests were answered by AI Agents, transferred to agents, or left incomplete. Includes metrics like conversation session trends, conversation path analysis, session distribution by channel, and AI Agent engagement broken down by self-service, drop-off, and agent transfer sessions.
- **Users Dashboard**: Central place for new and returning user trends. Provides total unique user count, returning user count, new user count, and weekly or daily user retention cohorts.
- **Performance Dashboard**: Helps understand AI Agent performance. Provides data on intents identified, tasks completed, and service or script failures.
- **Custom Dashboard**: Allows you to design custom reports and dashboards.
- **NLP Insights**: Provides in-depth insights into intent identification and task execution. Includes a summarized view of conversations, intents detected, and drop-offs presented as a flow diagram.
- **Task Execution Logs**: Provides in-depth insights into task execution data and AI Agent performance in completing tasks.
- **Conversations History**: Analyzes user interactions by summarizing key events including intent identifications, entity or confirmation retries, and task execution failures.
- **Conversation Flows**: Visualizes the journey of AI Agent-user interactions across conversation sessions for a given time period. Helps developers understand frequently used tasks, how users invoke them, popular task execution paths, and dropout points.
- **Conversation Insights**: Provides a visual map of user utterances to discover false positives and negatives and identify training opportunities. Uses deep neural models to form utterance clusters, each assigned a representative name. Supports tabular view and allows categorization of utterances as True Positives (TP), True Negatives (TN), False Positives (FP), and False Negatives (FN) for training.
- **Feedback Dashboard**: Uses the Feedback Survey feature to capture user feedback on products, services, and overall AI Agent experience. Supports NPS, CSAT, and other metrics. Displays survey results, score distribution trends, and overall score for a given period, with an option to export results.
- **LLM and GenAI Usage Logs**: Provides detailed insights into LLM and GenAI requests and responses. Helps AI Agent designers track feature usage, optimize prompts, and improve overall performance.
