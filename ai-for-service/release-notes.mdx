AI for Service receives updates every fifteen days with new features, enhancements, and bug fixes. Release schedules may be adjusted during holidays.

## Release Notes by Module

| Module | Description |
|---|---|
| [Automation AI](/ai-for-service/release-notes/automation-ai) | Feature updates and enhancements for DialogGPT, agent flows, agent node, prompts, and channel integrations. |
| [Search AI](/ai-for-service/release-notes/search-ai) | Updates to connectors, content extraction, retrieval strategies, answer generation, and analytics. |
| [Contact Center AI](/ai-for-service/release-notes/contact-center-ai) | Enhancements to the agent console, queue management, campaigns, routing, and supervisor tools. |
| [Agent AI](/ai-for-service/release-notes/agent-ai) | Updates to the Agent AI widget, agentic copilot, integration support, and real-time coaching. |
| [Quality AI](/ai-for-service/release-notes/quality-ai) | Updates to AutoQA, conversation mining, evaluation metrics, audit workflows, and reporting. |
| [Voice Gateway](/ai-for-service/release-notes/voice-gateway) | Enhancements to ASR/TTS providers, SIP trunk configuration, call control, and voice utilities. |
| [Platform Services](/ai-for-service/release-notes/platform-services) | Updates to LLM and generative AI framework, channels, app settings, admin console, and APIs. |
| [Deprecations](/ai-for-service/release-notes/deprecations) | Information on deprecated components, migration timelines, and upgrade guidance. |
