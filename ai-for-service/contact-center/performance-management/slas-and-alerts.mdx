SLAs & Alerts let you set acceptable thresholds for key performance indicators (KPIs)—such as Average Speed to Answer, Response Service Level, Abandonment Rate, and Transfer Rate-—across voice and chat interactions.

**Navigation:** Contact Center AI > Performance Management > SLAs & Alerts.

For details on Service Levels See, [Service Levels](/ai-for-service/contact-center/supervisor#service-levels)

---

## Service Level

### Create a Service Level

1. Select **+ New Service Level** in the upper-right corner.

   ![Add New Service Level](/ai-for-service/contact-center/performance-management/images/add-rule.png)

2. In the **New Service Level Rule** panel, enter a **Name** and **Description**.

   ![SLA Name & Description](/ai-for-service/contact-center/performance-management/images/sla-name-description.png)

3. Under **Configuration Setup**, select the **Queues** and **Channels**.

   ![Config Setup](/ai-for-service/contact-center/performance-management/images/config-setup.png)

4. Select a trigger.

   ![Triggers](/ai-for-service/contact-center/performance-management/images/choose-triggers.png)

   | Trigger | Screenshot |
   |---|---|
   | **Abandonment Rate** | ![Abandonment Rate](/ai-for-service/contact-center/performance-management/images/abandonment.png) |
   | **Average Speed to Answer (ASA)** | ![Average Speed to Answer](/ai-for-service/contact-center/performance-management/images/avg-speed.png) |
   | **Response Service Level (RSL)** | ![Response Service Level](/ai-for-service/contact-center/performance-management/images/rsl.png) |
   | **Transfer Rate** | ![Transfer Rate](/ai-for-service/contact-center/performance-management/images/transfer.png) |

5. Select **+ Add an action** and choose an action.

   ![Add Actions](/ai-for-service/contact-center/performance-management/images/action-options.png)

   | Action | Screenshot |
   |---|---|
   | **Alert** | ![Alert Config](/ai-for-service/contact-center/performance-management/images/alert-config.png) |
   | **Email** | ![Email Config](/ai-for-service/contact-center/performance-management/images/alert-email.png) |

6. Select **Save**. A confirmation message appears and the service level is created.

   ![Save SLA](/ai-for-service/contact-center/performance-management/images/save-sla-rule.png)

---

### Edit a Service Level

1. Select **Edit**.

   ![Edit SLA](/ai-for-service/contact-center/performance-management/images/edit-sla-button.png)

2. Make your changes and select **Save**. A confirmation message appears.

   ![Save Edited SLA](/ai-for-service/contact-center/performance-management/images/save-edit-sla.png)

### Edit a Service Level Name

1. Select **Edit**.

   ![Edit SLA](/ai-for-service/contact-center/performance-management/images/edit-sla-button.png)

2. Update the **Name** or **Description** and select **Save**. The updated name appears.

   ![Edit SLA Name](/ai-for-service/contact-center/performance-management/images/edit-sla-name-desc.png)

### Delete a Service Level

1. Select **Delete**.

   ![Delete SLA](/ai-for-service/contact-center/performance-management/images/delete-sla-rule.png)

2. Confirm by selecting **Delete**. A notification appears and the service level is deleted.

   ![Confirm Delete SLA](/ai-for-service/contact-center/performance-management/images/confirm-sla-delete.png)

## General Alerts

General Alerts let administrators and supervisors set up notifications for operational metrics and system events. For example, you can trigger alerts when users export:

- The **Interaction Details Report**
- Data from the **Interaction Dashboard**
- The **Interaction Details Report by Segment**

---

### Create a General Alert

1. Select the **General Alerts** tab, then select **+ New Alert** in the upper-right corner.

   ![Add Alert](/ai-for-service/contact-center/performance-management/images/add-alert-rule.png)

2. In the **New General Alert** panel, enter a **Name** and **Description**.

   ![Name and Description](/ai-for-service/contact-center/performance-management/images/new-general-alert-rule.png)

3. Under **Configuration Setup**, select a module and configure it as follows.

   ![Choose Module](/ai-for-service/contact-center/performance-management/images/module-selection.png)

#### After Call Work (ACW)

1. Select the **Queues**, **Channels**, and **Duration**.

   ![ACW Config Setup](/ai-for-service/contact-center/performance-management/images/config-setup.png)

2. Select **+ Add an action** and choose an action.

   ![Action Options](/ai-for-service/contact-center/performance-management/images/action-options.png)

   | Action | Screenshot |
   |---|---|
   | **Alert** | ![Alert](/ai-for-service/contact-center/performance-management/images/acw-alert.png) |
   | **Email** | ![Email](../performance-management/images/acw-email.png) |

3. Select **Save**. A notification appears and the alert is created.

#### Analytics > Interactions

1. Select a trigger.

   ![Choose Trigger](/ai-for-service/contact-center/performance-management/images/analytics-interactions-page.png)

2. Set the number of times the system exports data within the selected time interval.

   ![Export Settings](/ai-for-service/contact-center/performance-management/images/user-exports.png)

3. Select **+ Add an action** and choose an action.

   ![Action Options](/ai-for-service/contact-center/performance-management/images/action-options.png)

   | Action | Screenshot |
   |---|---|
   | **Alert** | ![Alert Config](/ai-for-service/contact-center/performance-management/images/alert-config.png) |
   | **Email** | ![Email Config](/ai-for-service/contact-center/performance-management/images/alert-email.png) |

4. Select **Save**. A notification appears and the alert is created.

   ![Saved Alert](/ai-for-service/contact-center/performance-management/images/analytics-interactions-page.png)

#### Voicemail in Agent Console

1. Select the **Queues**, number of **Voice Mails**, and **Time Interval**.

   ![Console Voicemail](/ai-for-service/contact-center/performance-management/images/console-voicemail-page.png)

2. Select **+ Add an action** and choose an action.

   ![Action Options](/ai-for-service/contact-center/performance-management/images/action-options.png)

   | Action | Screenshot |
   |---|---|
   | **Alert** | ![Alert Config](/ai-for-service/contact-center/performance-management/images/alert-config.png) |
   | **Email** | ![Email Config](/ai-for-service/contact-center/performance-management/images/alert-email.png) |

3. Select **Save**. A notification appears and the alert is created.
