The Kore.ai BotKit SDK is a Node.js library that sits between the Platform and your server, giving you full control over every message, webhook event, and agent transfer in your bot's conversation flow. Use it to inject custom business logic, integrate with third-party systems, handle live agent handoffs, and extend dialog tasks beyond what the standard API node supports.
