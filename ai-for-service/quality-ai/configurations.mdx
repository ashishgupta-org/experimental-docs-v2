### Metrics Measurement Types

Choose the right measurement type for each evaluation metric.

| Metric Type | Description |
|---|---|
| [By Question](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-question) | Evaluate agent responses to specific questions, statically or dynamically. |
| [By AI Agent](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-ai-agent) | Use AI agents to assess multiple conversation aspects in a single evaluation call. |
| [By Playbook Adherence](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-playbook-adherence) | Measure how well agents follow established procedures and workflows. |
| [By Dialog Task](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-dialog-task) | Evaluate adherence to predefined scripts and behavioral steps. |
| [By Value](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-value) | Validate agent accuracy against backend data using LLM-powered entity recognition. |
| [By Speech](ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-speech) | Assess voice behaviors such as crosstalk, dead air, and speaking rate. |
| [By Manual Evaluation](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-manual-evaluation) | Evaluates agent performance through human-led reviews when automated detection is insufficient. |
| [By Hold Etiquette](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-hold) | Evaluate how agents manage customer holds during voice interactions. |
| [By Transfer Etiquette](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-transfer) | Assess how agents handle internal customer transfers during voice interactions. |
| [AutoQA Prompting Guide](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/autoqa-prompting-guide) | Learn how to write effective prompts for LLM-based adherence detection. |
