Agent Scorecards let you define evaluation criteria at the individual agent level based on performance areas such as salesmanship and product knowledge. Each scorecard combines evaluation metrics into agent attributes. During call transfers, the system evaluates agents against all assigned scorecards-you can assign multiple scorecards per agent.

<Note>To view Agent Scorecards and Agent Attributes, enable the **Agent Scorecards** toggle in **Quality AI > Settings**.</Note>

Supervisors can have one of three permission levels for agent scorecards:

| Permission | Description |
|-----------|-------------|
| Full access | Create, edit, and manage scorecards and attributes |
| No access | Can't create or edit scorecards and attributes |
| View only | Read existing scorecards and attributes, no edits |

## Create a New Agent Scorecard

Navigate to **Quality AI** > **CONFIGURE** > **Agent Score Cards**.

1. Select **+ New Agent Score Card**.

   ![New Agent Score Card](/ai-for-service/quality-ai/configure/agent-scorecards/images/agent-scorecard-default-page.png)

### Settings

2. Select the **Settings** tab in the **Add New Agent Score Card** panel.

   ![Agent Scorecards Settings](/ai-for-service/quality-ai/configure/agent-scorecards/images/add-new-agent-score-card-settings.png)

3. Enter a **Name** for the scorecard.

4. Enter an optional **Description**.

5. Enable **Mark as Default** to assign this scorecard to all agents. Set the pass score (default: 40).

   <Note>There must always be an active default scorecard. The first scorecard you create is automatically set as default. To disable the default toggle, you must first assign another scorecard as default.</Note>

   ![Default Scorecard Alert](/ai-for-service/quality-ai/configure/agent-scorecards/images/default-scorecard-alert.png)

   ![Enable Default Scorecard](/ai-for-service/quality-ai/configure/agent-scorecards/images/enable-default-scorecard.png)

### Agent Attributes

6. Select the **Agent Attributes** tab.

   ![Add Agent Attributes](/ai-for-service/quality-ai/configure/agent-scorecards/images/agent-attributes.png)

7. Search and select one or more agent attributes.

8. Select **Add Agent Attributes**.

   ![Add Agent Attributes Weight](/ai-for-service/quality-ai/configure/agent-scorecards/images/add-agent-attributes-weight.png)

9. Assign a **Weight** to each selected attribute.

   <Note>If the scorecard is set as default, the agent leaderboard and dashboard display scores based on that scorecard.</Note>

### Assignments

10. Select the **Assignments** tab.

    ![Assignments](/ai-for-service/quality-ai/configure/agent-scorecards/images/assignments.png)

11. Search for agents and select **Add Agents** to assign them to the scorecard.

    ![Add Agents](/ai-for-service/quality-ai/configure/agent-scorecards/images/add-agents.png)

12. Search for agent groups and select **Add Groups** to assign at the group level.

    ![Add Agent Group](/ai-for-service/quality-ai/configure/agent-scorecards/images/add-agent-group.png)

13. Select **Create**. Enable the status toggle to activate agent-level scoring.

    ![New Scorecard](/ai-for-service/quality-ai/configure/agent-scorecards/images/newly-created-scorecard.png)

<Note>If a scorecard is set as default, the Assignments section is hidden and the scorecard is automatically assigned to all agents in the user's queues.</Note>

The scorecard list shows the following columns:

| Column | Description |
|--------|-------------|
| **Name** | Agent or agent group name |
| **Agent Attributes** | Attribute assigned to the scorecard |
| **Assignee** | User who assigned the scorecard |
| **Created By** | User who created the scorecard |
| **Pass Score** | Minimum score required to pass |
| **Status** | Active or inactive |

## Edit an Agent Scorecard

1. Select the scorecard and click the vertical ellipsis (⋮).

   ![Vertical Ellipsis Edit Button](/ai-for-service/quality-ai/configure/agent-scorecards/images/vertical-ellipsis-edit-button.png)

2. Select **Edit**.

   ![Edit Agent Scorecard](/ai-for-service/quality-ai/configure/agent-scorecards/images/edit-agent-scorecard.png)

3. Update the **Settings**, **Agent Attributes**, or **Assignments** as needed.

   ![Update Agent Scorecard](/ai-for-service/quality-ai/configure/agent-scorecards/images/update-agent-scorecard.png)

   <Note>If you enable the default toggle on another scorecard, the system replaces the current default after confirmation.</Note>

   ![Enable Default Scorecard](/ai-for-service/quality-ai/configure/agent-scorecards/images/enable-default-scorecard.png)

4. Select **Update** to save.

---

## Delete an Agent Scorecard

1. Select the scorecard and click the vertical ellipsis (⋮).

   ![Edit and Delete Agent Scorecard](/ai-for-service/quality-ai/configure/agent-scorecards/images/edit%20and%20delete-agent-scorecard.png)

2. Select **Delete**.

   ![Delete Agent Scorecard](/ai-for-service/quality-ai/configure/agent-scorecards/images/delete-agent-scorecard.png)

   <Note>You cannot delete the default scorecard. Assign another scorecard as default first. Deleting a scorecard removes all associated data.</Note>

## Agent Attributes

Agent Attributes define specific performance areas for agents — such as Professionalism, Product Knowledge, or Problem-Solving — by combining evaluation metrics. The system calculates an attribute score from the associated metric results.

Navigate to **Quality AI** > **Configure** > **Agent Attributes**.

![Agent Attributes](/ai-for-service/quality-ai/configure/agent-scorecards/images/agent-attributes-default-screen.png)

### Create a New Agent Attribute

1. Select **Agent Attributes** > **+ New Agent Attribute**.

   ![New Agent Attribute](/ai-for-service/quality-ai/configure/agent-scorecards/images/new-agent-attribute1.png)

2. In the **Add New Agent Attribute** panel:

   ![Add New Agent Attribute](/ai-for-service/quality-ai/configure/agent-scorecards/images/add-new-agent-attribute.png)

   - Enter a **Name**.
   - Enter an optional **Description**.
   - Search for and select evaluation metrics using **Add Evaluation Metrics**.

3. Select **Create**.

   ![Updated Attributes List](/ai-for-service/quality-ai/configure/agent-scorecards/images/added-new-attributes.png)

The attribute list shows:

| Column | Description |
|--------|-------------|
| **Name** | Attribute name |
| **Description** | Optional description |
| **Score Card** | Status of all linked scorecards |

### Edit an Agent Attribute

1. Select the attribute and click the vertical ellipsis (⋮).

   ![Edit Agent Attribute](/ai-for-service/quality-ai/configure/agent-scorecards/images/edit-attribute-list.png)

2. Select **Edit**.

   ![Edit Attribute Option](/ai-for-service/quality-ai/configure/agent-scorecards/images/edit-attribute-option.png)

3. Update the evaluation metrics and select **Update**.

### Delete an Agent Attribute

1. Select the attribute and click the vertical ellipsis (⋮).

   ![Delete Agent Attribute](/ai-for-service/quality-ai/configure/agent-scorecards/images/edit-attribute-list.png)

2. Select **Delete**.

   ![Delete Attribute Warning](/ai-for-service/quality-ai/configure/agent-scorecards/images/delete-agent-attribute.png)

   <Note>You cannot delete an attribute that is assigned to a scorecard. Remove it from all scorecards first. Deleting an attribute removes all associated data.</Note>
