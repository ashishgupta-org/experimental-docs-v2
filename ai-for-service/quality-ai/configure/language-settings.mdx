Language Settings lets administrators and supervisors configure languages for evaluation metrics and conversation insights across Quality AI.

Navigate to **Quality AI** > **CONFIGURE** > **Settings** > **Language Settings**.

![Language Settings](/ai-for-service/quality-ai/configure/settings/language-settings/images/lang-settings-page.png)

<Note>Only Administrators and Supervisors can enable or disable language settings at the app level.</Note>

---

## Add a New Language

1. Expand the **Language Settings** section.

   ![Language Settings Collapse Menu](/ai-for-service/quality-ai/configure/settings/language-settings/images/lang-settings-collapse-menu.png)

2. Select **+ Add Language**.

   ![Add Language](/ai-for-service/quality-ai/configure/settings/language-settings/images/settings-add-language.png)

3. Search for and select a language from the dropdown.

   ![Search to Add Language](/ai-for-service/quality-ai/configure/settings/language-settings/images/add-new-language.png)

   <Note>Supported languages: English, Dutch, Hindi, Japanese, German, and Spanish.</Note>

4. Enable the **Status** toggle to activate the language across the application (Auto QA, Scorecard, and Conversation Intelligence analytics).

5. Select **Save**.

---

## Set a Default Language

1. In the language list, select the language you want to set as default.

   ![Mark as Default](/ai-for-service/quality-ai/configure/settings/language-settings/images/mark-default.png)

2. Double-click the language to set it as the default.

   ![Change Default Language](/ai-for-service/quality-ai/configure/settings/language-settings/images/default-language-notification.png)

3. Select **Confirm**.

   <Note>Users with Auto QA permission can update the default language when multiple languages are configured. The first language added is the default. Changing the default only affects future interactions — existing data remains unchanged.</Note>

4. The selected language status changes to lighter mode to indicate it is the default.

   ![Default Language Status](/ai-for-service/quality-ai/configure/settings/language-settings/images/default-language-status-change.png)

5. Select **Save**.

---

## Disable a Language

1. Toggle off the **Status** switch for the language. A warning appears.

   ![Disable Language Warning](/ai-for-service/quality-ai/configure/settings/language-settings/images/disable-language.png)

2. Select **Confirm**.

   <Note>Disabling a language stops Auto QA, Scorecard, and Conversation Intelligence analytics for all conversations in that language.</Note>

3. Select **Save**.
