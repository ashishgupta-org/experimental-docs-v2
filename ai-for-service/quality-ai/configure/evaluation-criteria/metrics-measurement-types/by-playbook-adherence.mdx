The By Playbook Adherence metric measures and monitors how well contact center agents follow established procedures and workflows. It provides insights into process compliance, helping organizations maintain consistent service quality, ensure regulatory compliance, and identify improvement opportunities. Each metric is used for evaluation purposes and is assigned to a single agent attribute.

## Key Capabilities

| Capability | Description |
|-----------|-------------|
| **Automated adherence tracking** | Monitor compliance with entire playbooks or specific steps |
| **Configurable thresholds** | Set custom adherence percentages based on process criticality |
| **Flexible evaluation periods** | Configure time bounds for daily, weekly, or monthly assessments |
| **Granular analysis** | Evaluate adherence at the playbook, stage, or step level |
| **Performance insights** | Generate actionable data for coaching, training, and process optimization |

## Use Cases

- Agent performance evaluation — assess individual agent compliance
- Team performance analysis — compare adherence rates across teams
- Process compliance auditing — verify critical service steps are followed
- Training gap identification — spot playbook steps agents consistently struggle with
- Quality score calculation — incorporate adherence into overall quality assessments
- Coaching opportunities — identify areas needing additional support
- Operational reporting — generate management reports on process compliance

## Configure By Playbook Adherence Metric

1. Navigate to **Quality AI** > **Configure** > **Evaluation Forms** > **Evaluation Metrics**.
2. Select **+ New Evaluation Metric**.
3. From the **Evaluation Metrics Measurement Type** dropdown, select **By Playbook Adherence**.

   ![By Playbook Adherence Dropdown](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/images/by-playbook-dropdown.png)

4. Enter a descriptive **Name** for future audit reference.

5. Select a **Playbook Name** from the dropdown.

   ![Playbook Metrics Fields](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/images/by-playbook-fields.png)

6. Select an **Adherence Type**:

   **Entire Playbook:**
   - Evaluates adherence across the complete playbook.
   - Enter the minimum **Adherence Percentage** required. If adherence falls below this rate, the metric fails.

   **Steps:**
   - Evaluates adherence to a specific step within the playbook.
   - Select the **Stage** and **Step** for evaluation.

   ![Steps Configuration](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/images/by-playbook-adherence-type.png)

7. Select **Create**.

## Edit or Delete By Playbook Metric

1. Select the By Playbook Adherence metric and click the ellipsis (⋮).

   ![Edit Playbook Metric](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/images/playbook-modification.png)

2. Choose an option:
   - **Edit** to modify the metric.

     ![Edit Playbook](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/images/by-playbook-edit-metrics.png)

   - **Delete** to remove the metric.

3. Select **Update** to save changes.

### Deletion Requirements

Before deleting, confirm:

1. The metric isn't actively used in any evaluation form or workflow. The system shows a warning if it is.
2. Remove the metric from all associated evaluation forms.
3. The system allows deletion only after all dependencies are resolved.

![Playbook Metrics Delete Warning](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/images/playbook-metrics-delete.png)
