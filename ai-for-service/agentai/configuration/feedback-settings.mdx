Feedback Settings let you configure how agents provide feedback on the Agent AI widget. Two feedback types are available:

| Type | Description |
|------|-------------|
| **Context-Specific Feedback** | Configure up to six predefined negative feedback messages for Agentic Feedback, Dialog Tasks, Answers, FAQs, and Summaries. Up to five can be active at a time. Messages appear after agents select the thumbs-down icon. |
| **Overall Widget Feedback** | Configure up to six positive and six negative feedback messages. Up to five from each category can be active at a time. Disabled by default. |

---

## Configure Feedback Messages

The Feedback Settings section includes subsections for Agentic Feedback, Dialog Task Feedback, Answers Feedback, FAQ Feedback, Summary Feedback, and Overall Widget Feedback. Each subsection includes predefined messages that you can edit.

1. Sign in to XO Platform.
2. Go to **Agent AI** > **Configurations** > **Feedback Settings**.

   ![Feedback Settings](/ai-for-service/agentai/configuration/feedback-settings-images/feedback-settings.png)

3. Expand any subsection — **Agentic Feedback**, **Dialog Task Feedback**, **Answers Feedback**, **FAQ Feedback**, **Summary Feedback**, or **Overall Widget Feedback**.
4. Use the toggle to turn the subsection on or off.
   - For **Overall Widget Feedback**, expand **Positive** or **Negative Feedback** separately.

   ![Positive Negative Feedback Subsection](/ai-for-service/agentai/configuration/feedback-settings-images/positive-negative-feedback-subsection.png)

5. Select the edit icon on any feedback message.
6. Edit the message and press **Enter** to save.

---

## Turn On or Off Feedback Messages

Only active (turned on) feedback messages appear in the widget. Up to five of the six default messages can be active at a time.

1. Select the **Enable/Disable** toggle on a feedback message.
2. Select **Save**.

---

## Turn On or Off the Additional Comments Box

The **Display additional comment** box appears in the widget only if enabled here.

1. Select or clear the **Display additional comment** checkbox.

   ![Display Additional Comment Box](/ai-for-service/agentai/configuration/feedback-settings-images/display-additional-comment-box-enable-disable-5.png)

2. Select **Save**.

---

## Customize Feedback Language

The app's default language is pre-selected. You can change the display language for feedback messages.

1. Select the **language selection dropdown** next to the **Display additional comment** box.
2. Select a language and select **Save**.

   ![Change Feedback Message Language](/ai-for-service/agentai/configuration/feedback-settings-images/change-feedback-message-language-6.png)

---

## Feedback in the Agent AI Widget

Feedback messages and the additional comment box appear in the widget only when enabled. If not configured, only the thumbs up and thumbs down icons appear.

| Feedback Type | Trigger |
|---------------|---------|
| **Agentic Feedback** | When an Agentic Copilot Next Best Action suggestion is received. |
| **Dialog Task Feedback** | When a dialog task completes. |
| **Answers Feedback** | When an agent views an answer or web page. |
| **FAQ Feedback** | When an agent views an FAQ. |
| **Summary Feedback** | When an auto-generated summary appears. |
| **Overall Widget Feedback** | When the agent selects the ellipsis (more) icon at the widget's lower right. |

![Overall Widget Feedback Button](/ai-for-service/agentai/configuration/feedback-settings-images/overall-widget-feedback-button.png)

Agents can select multiple feedback messages and enter additional comments in the **Tell us more…** field (if **Display additional comment box** is enabled).

![Agentic Feedback Messages](/ai-for-service/agentai/configuration/feedback-settings-images/agentic-feedback-messages.png)

<Note>The **Submit** button appears only when feedback messages or the additional comment box are active in the widget.</Note>
