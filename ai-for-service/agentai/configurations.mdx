Use these configuration guides to set up and customize Agent AI for your deployment. The settings cover conversation behavior, widget appearance, external integrations, feedback collection, and language support.

## Configuration Guides

| Guide | Description |
|---|---|
| [Widget Settings](/ai-for-service/agentai/configuration/widget-settings) | Configure channel-specific widget behavior (Proactive Mode, Agentic Copilot, transfer handling) and global settings (URL behavior, data format, language, display options). |
| [Widget Theming and Layout Customization](/ai-for-service/agentai/configuration/widget-theming-layout-customization) | Customize the widget's visual layout — menu labels, tab order, search result display, dialog task grouping, and action button visibility — per channel. |
| [Conversation Events](/ai-for-service/agentai/configuration/conversation-events) | Configure Welcome Events (on-connect dialog tasks, greeting messages, priority settings) and Exit Events (end-of-conversation dialog tasks, conversation summaries, and Agentic App triggers). |
| [Feedback Settings](/ai-for-service/agentai/configuration/feedback-settings) | Configure context-specific and overall widget feedback messages for dialog tasks, answers, FAQs, summaries, and Agentic responses. |
| [Linked Services](/ai-for-service/agentai/configuration/linked-services) | Connect Agent AI to Search AI for enhanced search capabilities, and configure Agentic Copilot apps for autonomous, end-to-end conversation handling. |
| [Agent AI Mapping](/ai-for-service/agentai/configuration/agent-ai-mapping) | Assign Agentic Copilot apps and Dialog Tasks to specific queues so agents see only the tasks and automations relevant to their queue. |
| [Third Party Configuration](/ai-for-service/agentai/configuration/third-party-configuration) | Manage app configuration parameters such as `QueueIdentifier` and custom data to enable dynamic iframe URL generation for third-party CCaaS and CRM platforms. |
| [Custom LLM Model in GenAI](/ai-for-service/agentai/configuration/custom-llm-model-in-genai) | Create a custom LLM integration and select it as the model for generating conversation disposition summaries, replacing the default OOTB Kore Summarization model. |
| [Configure Auto Summarization with Custom Dialog Tasks](/ai-for-service/agentai/configuration/configure-auto-summarization-with-custom-dialog-tasks) | Host your own summarization model and connect it to Agent AI via a custom dialog task that fetches the transcript, generates the summary, and displays it at conversation end. |
| [Supported Languages](/ai-for-service/agentai/agent-experience/supported-languages) | Review which Agent AI features — Dialogs, Widget Localization, Greeting Messages, Summarization, Answers, Playbooks, and Coaching — are available for each supported language. |
