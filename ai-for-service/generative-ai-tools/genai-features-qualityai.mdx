Quality AI uses LLM-powered features to analyze agent-customer conversations. It evaluates agent performance, detects sentiment, identifies compliance gaps, and extracts actionable insights in real time and after the interaction.

## Features

### Advanced Topic Discovery based on Custom Taxonomy and Resolution Detection

Extracts topics and intents from agent conversations and detects resolution. Surfaces trends and anomalies based on customer sentiment, handle time, and resolution outcomes. [Learn more](/ai-for-service/quality-ai/analyze/topic-discovery)

### Agent Empathy Identification

Identifies instances of agent empathy in conversations where customers expressed negative sentiment. [Learn more](/ai-for-service/quality-ai/analyze/performance-insights).

### By Value Adherence Validation

Validates data adherence by comparing extracted metric values against reference data using multilingual LLM analysis. Returns binary adherence scores per metric. [Learn more](ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-value).

### By Value Metric Extraction

Extracts by-value metrics from user, AI Agent, and human agent messages. [Learn more](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-value).

### By Hold Adherence

Detects when an agent places a user on hold and resumes the conversation. [Learn more](/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-hold).

### By Transfer Adherence

Identifies whether the agent informs the customer about a transfer. [Learn more](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-transfer).

### Churn & Escalation Identification

Identifies customer churn risk and escalation intents across agent conversations. [Learn more](/ai-for-service/quality-ai/analyze/performance-insights).

### Conversation Phase Identification

Identifies conversation phases for phase-level sentiment trajectory and resolution-aware sentiment analysis. [Learn more](/ai-for-service/quality-ai/analyze/ai-assisted-manual-audit#sentiment-analysis).


### Crutch Word Usage Detection

Detects agent crutch word usage in customer conversations. [Learn more](/ai-for-service/quality-ai/analyze/performance-insights).

### Default Script Adherence

Determines agent adherence to default script steps: greeting, branding, privacy policy, customer verification, hold etiquette, and call closing. [Learn more](/ai-for-service/quality-ai/configure/conversation-intelligence#script-adherence-configuration).

### GenAI-based Agent Answer Adherence and Customer Trigger Detection

Verifies agent adherence and detects customer triggers without requiring utterance configuration or training. [Learn more](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-question).

### Generating Similar QM Utterance Suggestions

Generates semantically similar phrases during design time. [Learn more](/ai-for-service/quality-ai/configure/evaluation-criteria/metrics-measurement-types/by-question).

### Post Conversation Sentiment Analysis

Generates post-interaction sentiment and emotion moment insights for agent conversations. [Learn more](/ai-for-service/quality-ai/analyze/ai-assisted-manual-audit#by-ai-agent).

### Sentiment Analysis

Displays a real-time sentiment indicator during active chat conversations, updating dynamically as the conversation progresses. [Learn more](/ai-for-service/quality-ai/analyze/ai-assisted-manual-audit#by-ai-agent).

### Topic Modelling

Extracts popular topics and intents from agent conversations. [Learn more](/ai-for-service/quality-ai/analyze/topic-discovery).
