## Features

### Agent Response Rephrasing

Rephrases agent responses in the following tones:

| Tone | Description |
|------|-------------|
| **Formalize** | Makes the response suitable for business or formal contexts. |
| **Friendly** | Promotes an approachable and friendly tone. |
| **Expand** | Expands the response for formal settings. |
| **Rephrase** | Elevates formality for business or professional contexts. |

### Conversation Summary

Generates LLM-based conversation summaries at agent handoff, transfer between agents, and conversation wrap-up. [Learn more](../contactcenter/configurations/settings/llm-based-conversation-summary.md).

### Disposition Prediction for Agent Wrap-Up

Analyzes the full conversation transcript and disposition metadata, then recommends the most appropriate wrap-up code. Agents can accept or override the recommendation, improving accuracy and reducing wrap-up time. [Learn more](../console/interacting-with-customers.md#intelligent-disposition-code-suggestions).

### Sentiment Analysis

Displays a real-time sentiment indicator during active chat conversations, updating dynamically as the conversation progresses. [Learn more](../contactcenter/configurations/settings/real-time-sentiment-analysis.md).
