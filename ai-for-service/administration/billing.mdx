## Billing Sessions

Billing sessions are calculated based on two types:

**Usage Sessions (Bot Usage)**

- One usage session = 15 minutes of user conversation with the bot.
- Example: A 31-minute conversation counts as 3 sessions (0-15, 16-30, 31-end).
- Time spent with a human agent after transfer is excluded from bot session counts.

**Alert Sessions**

- Every alert delivered to a user counts as one alert session.
- Example: A sales report delivered every 8 hours = 3 alert sessions per day.

For more information, see [Product-specific Billing Sessions](/ai-for-service/manage-assistant/plan-and-usage/billing-and-payments#how-billing-works).

## Plan

Admin users can view and manage all plan subscriptions for their account and published apps.

- The Plan section lists all bots with their current status and balance.
- An **information** icon highlights plan changes such as downgrade notifications or expiring plans.
- Select the **Ellipses** icon to change the Bot Owner.

![Plan and Usage information](/ai-for-service/administration/images/billing-img2.png)

Hover over a line item to see the **Manage** option. From there you can:

- View the current plan and usage details
- Change the plan
- Add funds
- Request a plan extension

![Plan eligibility and usage](/ai-for-service/administration/images/billing-img3.png)

<Note>The **Billing > Plan and Usage** option does not appear in the Admin Console left menu for shared account users.</Note>

## Usage

The Usage section provides consolidated billing data across all apps in the workspace, with graphical and tabular views.

![Billing Graph](/ai-for-service/administration/images/admin-console-billing-graph.png)

![Billing Table](/ai-for-service/administration/images/admin-console-billing-table.png)

### Metrics

| Metric | Description |
|--------|-------------|
| **Billing Sessions** | Total completed sessions |
| **Voice Sessions** | Voice sessions consumed per app; calculated as every 15 minutes of interaction with the AI Agent or a human agent |
| **Agent Count** | Number of agents subscribed to the plan per app, aggregated by Day, Week, or Month |

### Filters

- Filter by individual app or view all apps
- Filter by date range: 24 hours, 7 days, 30 days, or custom range

### Tooltip

Hover over a data point to see:

- Date
- Sessions
- Voice Sessions
- Agents
