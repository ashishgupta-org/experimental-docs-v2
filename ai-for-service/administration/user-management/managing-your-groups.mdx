Use the **Groups** page in User Management to create and manage groups of users. Groups are useful when different departments or lines of business have distinct assistant requirements — assign developers and bots to groups accordingly. Groups created by admins are not visible to end users.

Groups can be created manually, imported from Active Directory, or created automatically when importing Active Directory organizational units.

![User Management - Groups](/ai-for-service/administration/user-management/images/bac-groups.png)

---

## Group Columns

| Column | Description |
|--------|-------------|
| **Group Name** | Name of the group. Select to edit name, description, and members. |
| **Description** | Optional description of the group. |
| **Source** | **Local** (manually created), **AD** (imported from Active Directory), or **System** (default group created by Kore.ai, such as *everyone*) |
| **Membership** | Number of users and child groups. Select the link to edit membership. |

Use the **Search** field to find groups — enter at least 3 characters for as-you-type results. Press **Enter** with the field cleared to show all entries.

<Note>Search uses up to the first 20 characters of each search term.</Note>

---

## Create a Group

1. In **User Management**, select **Groups**.
2. Select **+ New** to open the **Create New User Group** dialog.
3. On the **Common** tab:
   - Enter a **Group Name**.
   - Optionally enter a **Description**.

   ![Create New Group](/ai-for-service/administration/user-management/images/create-new-group.png)

4. On the **Members** tab, add users and/or groups:
   - Use **Sort**, **Filter**, or **Search** to find users and groups in the **Available Users/Groups** section.
   - Select one or more, then select the **→** arrow to move them to **Current Members**.
   - To add all available members at once, select the **⇒** double arrow.

   ![Add Users to Group](/ai-for-service/administration/user-management/images/bots-admin-user-mgmt-groups-create-new-group-add-users.png)

5. Select **Save**. A confirmation message appears.

---

## Modify a Group

1. In **User Management > Groups**, select the group name.
2. In the **Editing User Group** dialog:
   - On the **Common** tab, update the **Group Name** or **Description**.
   - On the **Members** tab, add or remove users and groups using the arrow controls.
3. Select **Save**.

![Modify Group](/ai-for-service/administration/user-management/images/bots-admin-user-mgmt-groups-edit.png)

---

## Delete a Group

<Note>Remove all users from a group before deleting it.</Note>

1. In **User Management > Groups**, select the checkbox for one or more groups.
2. On the **Action** bar, select the **Delete** icon.
3. In the confirmation dialog, select **Delete**.

![Delete Group](/ai-for-service/administration/user-management/images/bots-admin-user-mgmt-groups-delete.png)

> **Warning:** This operation is permanent and can't be undone.
