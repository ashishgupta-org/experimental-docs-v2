The User Management module lets admins manage users, groups, roles, and distribution lists. Access it from the Admin Console left navigation.

![User Management Module](/ai-for-service/administration/user-management/images/bac-users.png)

---

## Module Pages

| Page | Description |
|------|-------------|
| [User Management](/ai-for-service/administration/user-management/managing-your-users) | View and manage user roles, status, and message activity. Configure user profile fields and edit permissions. |
| [Groups Management](/ai-for-service/administration/user-management/managing-your-groups) | Create and manage groups of users, including groups imported from Active Directory. |
| [Role Management](/ai-for-service/administration/user-management/role-management) | Create and assign admin and bot roles to organize the developer community. |
| **Distribution Lists** | View distribution lists from Microsoft Active Directory, including user counts for managed and unmanaged users. Requires Active Directory configuration. |
