The User Management module lets admins manage users, groups, roles, and distribution lists. Access it from the Admin Console left navigation.

![User Management Module](../images/bac-users.png)

---

## Module Pages

| Page | Description |
|------|-------------|
| [User Management](./managing-your-users.mdx) | View and manage user roles, status, and message activity. Configure user profile fields and edit permissions. |
| [Groups Management](./managing-your-groups.mdx) | Create and manage groups of users, including groups imported from Active Directory. |
| [Role Management](./role-management.mdx) | Create and assign admin and bot roles to organize the developer community. |
| **Distribution Lists** | View distribution lists from Microsoft Active Directory, including user counts for managed and unmanaged users. Requires Active Directory configuration. |
