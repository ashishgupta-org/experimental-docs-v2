Use the **Bulk Invite** page to invite a large number of users at once by uploading a text file of email addresses.

![Bulk Invite](../images/inviting-many-users-img1.png)

---

## Before You Start

- The import file must be a `.txt` file with one email address per line, or a comma-separated list.
- Entries with invalid email formats are ignored.
- Invitation emails are sent to each successfully imported address.
- Existing account users are not sent invitations.

---

## Step 1 — Create the Invite File

1. Open a text editor.
2. Enter one email address per line:

   ```
   john.doe@example.com
   jane.doe@example.com
   ```

3. Save the file as a `.txt` file (for example, `KoreInvite.txt`).

![Sample Invite File](../images/inviting-many-users-img2.png)

---

## Step 2 — Upload the File

1. In the **Enrollment** module, select **Bulk Invite**.
2. Select **Choose File** and open your invite file.
3. Select **Send Invitations**.

The import status appears below the button.

4. Select **View details here** to review per-user import results.
5. Select **File Based Bulk User Invite History** to view results from prior imports.

![Upload Details](../images/inviting-many-users-img3.png)

---

## View Current Import Results

1. On the **Bulk Invite** page, select **View details here**.
2. The **Report on File Based Bulk User Invite** dialog opens.

   ![Import Report](../images/inviting-many-users-img4.png)

3. Select **Invite History** to view results from a previous import date.

---

## View Import History

1. On the **Bulk Invite** page, select **File Based Bulk User Invite History**.
2. The **History of Bulk User Invite** dialog opens.

   ![Invite History](../images/inviting-many-users-img5.png)

3. In the **Report** column, select **Results** for the import you want to review.

   ![Import Report Details](../images/inviting-many-users-img4.png)
