The Analytics module provides a history of all administrative activity. Use it to review admin and user events such as logins, logouts, and configuration changes, and to view bot chat history.

---

## Audit Report

The Audit Report page shows system and user events with the following attributes: **User Name**, **User Email**, **Date/Time**, and **Description**.

![Analytics - Audit Reports](../images/analytics-img1.png)

### Filters

Use filters to narrow the event list:

**Date Filter**

| Option | Description |
|--------|-------------|
| Past 24 hours | Default selection |
| Past 7 days | Last week's events |
| Custom range | Select start and end dates |

**Event Filter** (select via the **Filters** button)

| Filter | Description |
|--------|-------------|
| User Name / App Name | Show events by a specific user or app |
| Event Category | Filter by category — shows All by default |
| Event Name | Filter by a specific event within the selected category |

![Event filter options](../images/analytics-img2.png)

### Search

Enter at least 3 characters in the **Search** field to filter results as you type. Clear the field and press **Enter** to show all entries.

<Note>Search uses up to the first 20 characters of each search term.</Note>

### Export

Select the **Export** icon to download the audit report as a CSV file.

### Record Details

Select any entry to view full event details.

![Record details](../images/analytics-img3.png)

![Export search](../images/analytics-img4.png)

<Note>The Audit Report includes events from multiple Kore.ai products. To filter for XO Platform data, search for `App Builder` (case-sensitive) in the Description field. For complex filtering, export the report and use a spreadsheet editor.</Note>

## Bot Chat History

View chat history for all deployed bots from **Analytics > Bot Chat History**.

![Analytics - Bot Chat History](../images/analytics-img5.png)

Select a chat entry to view the full conversation. You can navigate to the user's previous and next sessions.

### Columns

| Column | Description |
|--------|-------------|
| Bot Name | Name of the bot in the chat |
| Timestamp | Date and time of the chat (YYYY-MM-DD HH:MM AM/PM) in the selected time zone |
| Channel | Channel where the chat occurred |
| Language | Language of the chat |
| User | Platform-generated user ID or channel-assigned user identity |
| Export | Option to export the individual chat in Excel or PDF format |

### Filters and Controls

**Time Zone** — Defaults to *Local*. Change using the **Change** link.

**Search** — Enter at least 3 characters to filter results. Clear and press **Enter** to show all.

**Timestamp Filter**

| Option | Description |
|--------|-------------|
| 24 hours | Default setting |
| 7 Days | Last week |
| Custom | Select start and end dates |

**Export** — Download Chat History results in CSV format.

**Filters**

| Filter | Description |
|--------|-------------|
| Bot | Filter by one or more bots |
| User | Filter by Kore-generated or channel user ID (available only when a single bot is selected) |
| Conversation type | Interactive sessions (user sent messages) or non-interactive sessions |
| Channel | Filter by interaction channel |
| Language | Filter by interaction language |
| Meta tags | Filter by message, user, or session-level meta tags (AND condition across multiple tags; available when at least one bot is selected) |

![Export file of Chat History](../images/analytics-img8.png)

## Chat History Details

Selecting a chat entry opens detailed user information.

![Chat History details](../images/analytics-img9.png)

<Note>Trace IDs are retained in logs for 30 days. Expired Trace IDs show: `Trace Id: Trace records for this message are not available`.</Note>

### User Profile

| Attribute | Description |
|-----------|-------------|
| Kore User ID | Platform-assigned user ID |
| Channel Data | User context data from the channel |
| User Meta Tags | Count and key-value pairs of the most recent meta tags |
| Latest Interaction | Last interaction timestamp |
| Total Conversation Sessions | All-time count of interactive and non-interactive sessions |
| Total Sessions (Last 30 Days) | Sessions in the last 30 days |
| Last 30 Days Intent Detection Rate | (Identified intents / (Identified + Unidentified)) × 100 |
| Intents Requested | Total identified + unidentified utterances |
| Intents Identified | Total identified intents |
| Last 30 Days Goal Completion Rate | (Success tasks / (Success + Failed)) × 100 |
| Tasks Initiated | Total success + failed tasks |
| Tasks Completed | Successfully completed tasks |
| Recent Conversation Flows | Top 10 flows executed in the last 30 days |

### User Conversation Sessions

**Session Attributes**

| Attribute | Description |
|-----------|-------------|
| Session Start | Session start date and time |
| Session End | Session end date and time |
| Channel | Channel of the session |
| Agent Transfer Tag | Sessions where the user was transferred to an agent |
| Drop Off Tag | Sessions where the user dropped off |
| Total Success Tasks | Count of successfully completed tasks |
| Total Failed Tasks | Count of failed tasks |
| Intents Identified | Count of identified intents |
| Intents Not Identified | Count and list of unidentified intents |
| Conversation Path | Series of tasks initiated in the session |
| Session Meta Tags | Count and details of session meta tags |

**Conversation Transcript**

| Attribute | Description |
|-----------|-------------|
| Message Meta Tags | Chat transcript annotated with message-level tags |
| Agent Transfer | Marks the point of agent transfer |
| Drop Off | Marks the point the user dropped off |
