---
title: End Flow Node
---

Use this node to define the message played or displayed to the customer when the flow ends.

## Configure the Node

### General Settings

In **General Settings**, configure the following:

![General Settings](/ai-for-service/flows/node-types/images/general-settings-end-flow.png)

1. **Custom Name**: Enter a name to identify the node.
2. **Your Message to User**: Select a message type under **Choose your message type**:

    * **Text to Speech**: Type or select a message to convert to speech. Click **+ Create New** to add a new prompt, or **+ Add Another** to define multiple messages (one is randomly selected at runtime).

        ![Text to Speech](/ai-for-service/flows/node-types/images/text-to-speech-end-flow.png)

    * **Audio File**: Type a message or click **+ Upload New** to upload an .mp3 or .wav file.

        ![Audio File](/ai-for-service/flows/node-types/images/audio-file-end-flow.png)

    * **Link**: Click **+ Create New** to add message text and an audio file URL (.mp3 or .wav). Click **+ Add Another** for additional URLs.

        ![Link](/ai-for-service/flows/node-types/images/link-end-flow.png)

3. **Go to node**: Select the node for error handling. You can create a new node or select an existing one.
