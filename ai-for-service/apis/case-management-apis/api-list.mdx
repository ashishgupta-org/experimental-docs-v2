The following table lists the APIs available for Case Management, grouped by the functionality they address.

## Case Management APIs

| API | Description | Associated App Scope |
|-----|-------------|----------------------|
| [Create Case](/ai-for-service/apis/case-management-apis/create-a-case) | Create a new case in the case management system. | Case Management Configuration |
| [Get Cases](/ai-for-service/apis/case-management-apis/get-cases) | Retrieve case data with filtering, sorting, and pagination options. | Case Management Configuration |
| [Update Case](/ai-for-service/apis/case-management-apis/update-a-case) | Update an existing case in the case management system. | Case Management Configuration |
| [Create Task](/ai-for-service/apis/case-management-apis/create-a-task) | Create a new task within the case management system. | Case Management Configuration |
| [Get Tasks](/ai-for-service/apis/case-management-apis/get-tasks) | Retrieve task data with filters for status, priority, assignment, channels, and date ranges. | Case Management Configuration |
| [Update Task](/ai-for-service/apis/case-management-apis/update-a-task) | Update an existing task in the case management system. | Case Management Configuration |
