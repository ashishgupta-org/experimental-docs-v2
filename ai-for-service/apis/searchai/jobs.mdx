This API returns the details of a job. The response varies depending on the job type.

| Field | Value |
|---|---|
| **Method** | GET |
| **Endpoint** | `<host_url>/api/public/bot/<AppID>/search/jobs/<JobID>` |
| **Content-Type** | `application/json` |
| **Authorization** | `auth: <JWT Token>` |
| **API Scope** | Answer Generation |

## Query Parameters

| Parameter | Description | Mandatory |
|---|---|---|
| `AppID` | Provide your application ID here. | Yes |
| `JobID` | Unique identifier for the job. | Yes |
