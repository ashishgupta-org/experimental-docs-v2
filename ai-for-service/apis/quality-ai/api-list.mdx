The following table lists the APIs available for Quality AI.

| API | Description | App Scope |
|-----|-------------|-----------|
| [Raw Data API](/ai-for-service/apis/quality-ai/raw-data-api) | Provides detailed Quality AI and Conversation Intelligence data, including quality metrics and analytics for each interaction | Configuration |
| [Auto QA Reporting API](/ai-for-service/apis/quality-ai/auto-qa-reporting-api) | Provides interaction-level Auto QA scoring and metric qualification based on the associated Quality AI form | Configuration |
