Deletes a specific execution result of a Batch Test Suite.

| Property | Value |
|---|---|
| **Method** | DELETE |
| **Endpoint** | `https://{host}/api/public/bot/{botId}/testsuite/{testSuiteName}/{testRunId}` |
| **Content Type** | `application/json` |
| **Authorization** | `auth: {{JWT}}` — See [How to generate the JWT Token](/ai-for-service/apis/automation/api-introduction/#generating-the-jwt-token). |
| **API Scope** | App Builder: Batch Tests Management; Admin Console: Batch Tests Management |

## Query Parameters

| Parameter | Description | Required |
|---|---|---|
| `host` | The environment URL. For example, `https://platform.kore.ai` | Required |
| `BotID` | Bot ID or Stream ID, accessible under **General Settings** on the App Builder. | Required |
| `testSuiteName` | Name of the test suite on the App Builder. | Required |
| `testRunId` | The unique identifier of an execution result obtained by running the [test execution API](/ai-for-service/apis/automation/batch-test-execution). | Required |

## Sample Request

```json
curl --location --request DELETE \
     'https://{host}/api/public/bot/{botId}/testsuite/{testSuiteName}/{testRunId}' \
      --header 'auth: {YOUR_JWT_ACCESS_TOKEN}' \
      --header 'bot-language: {language-code}'
```

## Body Parameters

No body parameters are passed.

## Sample Response

```json
{
    "message": "Test Result Removed Successfully"
}
```
