AI for Service exposes a comprehensive set of REST APIs across its product modules, enabling you to build integrations, automate workflows, and extract insights programmatically. Use the links below to navigate to the API reference for each module.

| Module | Description |
|--------|-------------|
| [Automation AI](/ai-for-service/apis/automation/api-list) | APIs for bot lifecycle management, NLP training, analytics, admin operations, and data table access. |
| [Contact Center AI](/ai-for-service/apis/contact-center/api-list) | APIs for agent and queue management, analytics, campaign operations, and voice gateway capabilities. |
| [Agent AI](/ai-for-service/apis/agent-ai/api-list) | APIs for retrieving conversation data, controlling transcription sessions, and triggering post-conversation hooks. |
| [Quality AI](/ai-for-service/apis/quality-ai/api-list) | APIs for accessing raw Quality AI data and Auto QA scoring reports at the interaction level. |
| [Search AI](/ai-for-service/apis/searchai/api-list) | APIs for answer generation, content and chunk management, data ingestion, connectors, and app import/export. |

<Note>
All APIs are subject to rate limits. If a rate limit is exceeded, the API call fails with: "Rate limit for this API has been reached. Please try again after some time." For details, see [Rate Limits](/ai-for-service/apis/rate-limit).
</Note>
