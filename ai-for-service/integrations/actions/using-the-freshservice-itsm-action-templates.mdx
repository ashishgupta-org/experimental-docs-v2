Use prebuilt Freshservice action templates to auto-create dialog tasks.

**Prerequisites:** Configure [Freshservice](configuring-the-freshservice-action.md) and install templates before proceeding.

To access templates:
1. Go to **Automation AI > Use Cases > Dialogs** and click **Create a Dialog Task**.
2. Under **Integration**, select **Freshservice** to view action templates.

   ![View Freshservice action template](/ai-for-service/integrations/actions/images/freshservice-action-img11-tem-img2.png)

If no integration is configured, click **Explore Integrations** to go to the Actions page.

![Explore integrations](/ai-for-service/integrations/actions/images/freshservice-tem-img3.png)

---

## Supported Actions

| Action | Description | Method |
|---|---|---|
| Create a Ticket | Creates a ticket in Freshservice | POST |
| Get Ticket by ID | Fetches ticket details by ID | GET |
| List All Tickets | Retrieves all tickets | GET |
| Update a Ticket | Updates a ticket | PUT |
| Search Ticket by Field | Searches tickets by field | GET |
| Delete a Ticket | Deletes a ticket | DELETE |

## Create a Ticket

1. Install the template from [Freshservice Templates](configuring-the-freshservice-action.md#step-2-install-freshservice-action-templates).
2. The _Create Ticket_ dialog task is added with:

   ![Create Ticket dialog](/ai-for-service/integrations/actions/images/freshservice-tem-img4.png)

   - **createTicket** - User intent to create a ticket.
   - **subject**, **description**, **email**, **priority**, **status**, **phone** - Entity nodes for ticket details.
   - **createTicketScript** - Bot action script to prepare the request.
   - **createTicketService** - Bot action service to create the ticket. Click **Edit Request**:

     ![Edit request](/ai-for-service/integrations/actions/images/freshservice-tem-img5.png)

     **Sample Request:**
     ```json
     {
       "email": "john.doe@example.com",
       "subject": "Support payment",
       "description": "Payment is pending issue",
       "status": 2,
       "priority": 3
     }
     ```

     **Sample Response:**
     ```json
     {
       "ticket": {
         "id": 57,
         "subject": "Support payment...",
         "description": "Payment is pending issue",
         "status": 2,
         "priority": 1,
         "type": "Incident",
         "created_at": "2022-09-28T07:00:14Z"
       }
     }
     ```

   - **createTicketMessage** - Message node to display the result.

3. Click **Train**, then **Talk to Bot** to test.
4. Follow prompts to create a ticket.

   ![VA prompt](/ai-for-service/integrations/actions/images/freshservice-tem-img6.png)

## Get Ticket by ID

1. Install the template from [Freshservice Templates](configuring-the-freshservice-action.md#step-2-install-freshservice-action-templates).
2. The _Get Ticket by ID_ dialog task is added with:

   ![Get Ticket By ID dialog](/ai-for-service/integrations/actions/images/freshservice-tem-img7.png)

   - **getTicketbyId** - User intent to find a ticket by ID.
   - **ticketID** - Entity node for the ticket ID.
   - **getTicketbyIdService** - Bot action service to find the ticket. Click **+Add Response**:

     ![Add response](/ai-for-service/integrations/actions/images/freshservice-tem-img8.png)

     **Sample Response:**
     ```json
     {
       "ticket": {
         "id": 57,
         "subject": "Support payment...",
         "description": "Payment is pending issue",
         "status": 2,
         "priority": 1,
         "type": "Incident"
       }
     }
     ```

   - **getTicketbyIdMessage** - Message node to display the result.

3. Click **Train**, then **Talk to Bot** to test.
4. Follow prompts to find a ticket by ID.

   ![VA prompt](/ai-for-service/integrations/actions/images/freshservice-tem-img9.png)

5. You can now view the ticket in Freshservice.

## Update a Ticket

1. Install the template from [Freshservice Templates](configuring-the-freshservice-action.md#step-2-install-freshservice-action-templates).
2. The _Update a Ticket_ dialog task is added with:

   ![Update a Ticket dialog task](/ai-for-service/integrations/actions/images/freshservice-tem-img13.png)

   - **updateTicket** - User intent to update a ticket.
   - **ticketID**, **updateField**, **email**, **subject**, **description**, **status**, **priority** - Entity nodes.
   - **updateTicketScript** - Bot action script for the update.

     ![View component properties](/ai-for-service/integrations/actions/images/freshservice-tem-img14.png)

   - Click **Edit Request**:

     ![Edit request](/ai-for-service/integrations/actions/images/freshservice-tem-img15.png)

     **Sample Request:**
     ```json
     {
       "email": "john.doe@example.com",
       "subject": "LOGIN ISSUE...",
       "description": "Unable to login...",
       "status": 4,
       "priority": 2
     }
     ```

     **Sample Response:**
     ```json
     {
       "ticket": {
         "id": 54,
         "subject": "LOGIN ISSUE...",
         "description": "Unable to login...",
         "status": 4,
         "priority": 2
       }
     }
     ```

   - **updateTicketMessage** - Message node to display the result.

3. Click **Train**, then **Talk to Bot** to test.
4. Follow prompts to update a ticket.

   ![VA prompt](/ai-for-service/integrations/actions/images/freshservice-tem-img16.png)

## Search Ticket by Field

1. Install the template from [Freshservice Templates](configuring-the-freshservice-action.md#step-2-install-freshservice-action-templates).
2. The _Search Ticket by Field_ dialog task is added with:

   ![Search Ticket by Field dialog task](/ai-for-service/integrations/actions/images/freshservice-tem-img17.png)

   - **searchTicket** - User intent to search tickets.
   - **chooseField**, **priority**, **email**, **status** - Entity nodes for search criteria.
   - **searchTicketScript** - Bot action script to prepare the search.
   - **searchTicketService** - Bot action service to search tickets. Click **+Add Response**:

     ![Add response](/ai-for-service/integrations/actions/images/freshservice-tem-img18.png)

     **Sample Response:**
     ```json
     {
       "tickets": [
         { "subject": "refer", "id": 55, "priority": 3, "status": 5 },
         { "subject": "Support payment...", "id": 7, "priority": 3, "status": 2 }
       ],
       "total": 2
     }
     ```

   - **searchTicketMessage** - Message node to display results.

3. Click **Train**, then **Talk to Bot** to test.
4. Follow prompts to search tickets.

   ![VA prompt](/ai-for-service/integrations/actions/images/freshservice-tem-img19.png)
