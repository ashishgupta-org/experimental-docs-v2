The platform offers integration capabilities to connect your AI Agent with business applications, external NLU engines, and agent handoff systems. Use pre-built integrations and low-code tools to exchange data, enable agent transfers, and leverage external NLU models.

---

## Integration Types

- **Actions**: Pre-built integrations with popular business applications (ServiceNow, Salesforce, Zendesk, HubSpot, and more). Each integration includes a full conversation flow definition, a low-code API interface, pre-built response mapping, and dialog task templates. [Learn more](../integrations/actions/actions.md)

- **Agent Transfer**: Configure popular integrations to hand over conversations seamlessly without BotKit. These integrations are hosted by the platform — no custom BotKit hosting required. [Learn more](../integrations/agents/agent-transfer-integrations.md)

- **External NLU**: Co-exist an external NLU engine with the platform's built-in NLP engines. Lets you use the NLU engine of your preference while keeping all other platform features. [Learn more](../integrations/external-nlu/external-nlu-adapters.md)
