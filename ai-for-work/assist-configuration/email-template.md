# Email Templates

Email Templates allows you to customize invitation emails that you send to users when you add them to the account, workspace, or specific applications. These templates provide a consistent communication experience while allowing you to tailor messages to your organization's needs.

You can use the Email Templates feature to:

* Customize invitation emails for different user roles and applications.
* Enable or disable email notifications for specific invitation types.
* Preview emails before you send them to invited users.
* Maintain consistent messaging across all applications that you provision to your account.

## Accessing Email Templates

To access Email Templates:

1. Navigate to **Account Hub** from the top navigation bar.
2. In the left sidebar, expand **Email Templates.**
3. Select the template category you want to configure:
    * **Admin Templates** - for account-level administrative invitations.
    * Application-specific templates - for application member invitations.

### Admin Templates

Admin Templates manage account-level invitations and include:

**Admin invitation template**: The system sends this email template when you add a user as an Admin to the account. Use this template to communicate administrative privileges and responsibilities.

**Custom Admin invitation template**: The system sends this email template when you assign a user a custom administrative role. Customize this template to explain the specific permissions you grant to the custom role.

**Workspace invitation template**: The system sends this email template when you invite a user to join a specific workspace. This template helps you onboard members in project-specific environments.


### Application Templates

Each provisioned application has its own email template section.

**Member invitation template**: This system added this email template sent when ypu add a user as a member to the AI for Work application.

Similar templates exist for all other provisioned applications in your account.


## Customizing Email Templates

For each template, you can:

1. **Enable or disable** the template using the toggle switch on the right side.
2. **Click on the template** to open the editor.
3. **Edit the subject line** to match your organization's tone.
4. **Customize the email body** while keeping essential information intact.
5. **Preview the email** to see how it will appear to the recipient.
6. **Include links** to relevant resources (such as the Admin Console or application-specific guides) for easy access.


## Manage Templates

* The system organizes templates by application and shows all applications that you provision to your account.
* The system provides default subject and body text for your convenience, but you can modify them as needed.
* Changes that you make to templates apply to all future invitation emails of that type.
* Admins and Custom Admins can make final adjustments to ensure templates meet organizational standards before the system sends invitations.
* Each application maintains its own separate member invitation template.

The email template includes a link to the Admin Console, ensuring that the invited member can easily access the necessary resources.
