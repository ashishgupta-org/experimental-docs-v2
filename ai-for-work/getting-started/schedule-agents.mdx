The **Scheduled Agents** feature enables you to automate agent execution at predefined intervals — running agents automatically without manual intervention to streamline recurring tasks and workflows.

```
Configure Scheduler → Set Frequency → Define Instructions → Monitor Executions
```

---

## Accessing Scheduled Agents

1. Log in to your Platform account.
2. Navigate to the **Agents and Flows** section to view all available agents.
3. Open an individual agent to access its scheduler settings.
    ![Scheduled Agents](./images/scheduled-agents-1.png "Scheduled Agents")

---

## Configuring a Scheduler

Open an individual agent to access its dedicated scheduler settings panel.

    ![Configuring Scheduler](./images/scheduled-agents-2.png "Configuring Scheduler")

Click the **Edit** button to open the agent's scheduling details. You can configure:

1. **Allow Schedule Trigger for Agent** — Enable or disable scheduling for this agent.
2. **Default Schedule** — Set the frequency and timing: Once, Hourly, Daily, Weekly, Monthly, Custom, or Cron.
3. **Instructions to Agent** — Define what the agent should execute when the schedule runs.
4. **Behavior** — Configure notifications to control how the system communicates with users about scheduled executions.

Use the toggle to enable or disable the scheduler at any time without changing the configuration.

    ![Enable disable scheduler](./images/scheduled-agents-3.png "Enable disable scheduler")

---

## Managing All Schedulers

The platform provides a centralized view for managing all schedulers across your agents and workflows. Navigate to the **Scheduler Agents** section to view, edit, and monitor all configured schedulers in one location.

![Managing All Schedulers](./images/scheduled-agents-4.png "Managing All Schedulers")

**Key capabilities:**

* Review all active and inactive schedulers.
* Modify scheduler configurations.
* Monitor scheduler execution history.
* Manage multiple schedulers efficiently.

**Use when**: You need to audit, update, or monitor scheduled automations across multiple agents from a single location.

---

## Updating an Existing Schedule

1. Select the agent from the **Schedule Agents** list.
2. The agent's scheduling details panel opens.

    ![Scheduling details](./images/scheduled-agents-5.png "Scheduling details")

3. Modify any of the following settings:
    - Schedule trigger status (enable/disable)
    - Schedule frequency and timing
    - Agent instructions
    - Notification preferences
4. Save your changes.
