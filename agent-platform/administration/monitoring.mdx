The Monitoring module in Agent Platform tracks events, actions, and changes across your account. Use it to maintain security, ensure compliance, and optimize performance across roles, integrations, models, tools, users, datasets, and guardrails.

| Page | Description |
|---|---|
| [**Audit Logs**](/agent-platform/administration/monitoring/audit-logs) | Track every user action and system event in your account, including logins, role changes, model updates, and workflow tool activities. Filter, search, and export logs for compliance and security review. |
| [**Analytics**](/agent-platform/administration/monitoring/analytics) | Monitor performance metrics and trends across workflow tools and models, including success rates, response times, and resource usage. |
| [**Model Traces**](/agent-platform/administration/monitoring/model-traces) | Review run-level details for any model deployed or integrated in your account, including request inputs, generated outputs, response times, and run metadata. |
| [**Custom Scripts**](/agent-platform/administration/monitoring/monitor-custom-scripts) | Monitor execution status and logs for all custom scripts deployed in your account, including run history, response times, and error details. |
